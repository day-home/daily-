# 하루 원판

일상홈피용 원형 24시간 플래너 초안입니다.

## 파일 구성

- `index.html` : 홈페이지 뼈대
- `styles.css` : 디자인
- `app.js` : 원형 플래너 기능
- `.nojekyll` : GitHub Pages에서 그대로 배포하기 위한 빈 파일

## 사용법

1. `index.html`을 브라우저로 열어봅니다.
2. 원판을 누르고 드래그해서 시간을 고릅니다.
3. 제목과 색을 정합니다.
4. `원판에 추가하기`를 누릅니다.
5. 같은 제목의 시간은 `오늘의 합계`에 자동으로 더해집니다.

일정은 서버가 아니라 브라우저의 localStorage에 저장됩니다.
즉, 같은 브라우저에서는 남아 있지만 다른 기기에서는 공유되지 않습니다.
